# Contributing Guidelines

Dự án này hoan nghênh mọi đóng góp nhằm cải thiện chất lượng và tính năng.

## Quy trình đóng góp
1. Fork repository
2. Tạo branch mới từ `main`
3. Commit thay đổi với mô tả rõ ràng
4. Push branch lên GitHub
5. Tạo Pull Request

## Quy tắc
- Code rõ ràng, dễ đọc
- Không commit thư mục build
- Tuân thủ cấu trúc project
