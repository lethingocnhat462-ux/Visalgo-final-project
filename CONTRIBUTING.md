Reporting Bugs

If you encounter any issues while using the visualizer (e.g., incorrect sorting logic or UI glitches), please open an Issue on GitHub. Provide as much detail as possible, including:



A clear title and description.

Steps to reproduce (including the input array used).

Screenshots of the error.

Suggesting Enhancements

We are always looking for new features! You can suggest:



New algorithms (e.g., Quick Sort, Merge Sort).

Speed control improvements.

New UI themes and colors.

ğŸ”„ Pull Request Process

Fork the repository and create your branch from main.

Ensure your code follows the existing folder structure:

Algorithms: src/algorithms/.

Components: src/components/.

If you've added an algorithm, ensure it returns a steps array for the Visualizer component to process.

Submit a Pull Request with a clear description of your changes.

JavaScript

Color Coding Convention

To maintain UI consistency, please use the established color states in your Bar components:



Default: #3498db (Blue).

Comparing: #e74c3c (Red) or Yellow.

Swapping: Highlighted Red.

Sorted: #2ecc71 (Green).
